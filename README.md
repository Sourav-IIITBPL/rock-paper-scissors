# Rock Paper Scissors

A simple Rock Paper Scissors game implemented using vanilla JavaScript,
HTML, and CSS.

## Overview
This project was built in March 2025 while learning JavaScript fundamentals.
It focuses on conditional logic, randomization, and basic user interaction
in the browser. The project is pushed later for record-keeping and reference.

## Features
- User vs computer gameplay
- Randomized computer choice
- Immediate result display after each round

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
Open `index.html` in any modern web browser.
